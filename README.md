# Smart Email Writer ✉️

**Smart Email Writer** is an AI-powered tool that converts short ideas into full professional emails instantly. Perfect for saving time and improving communication.

---

## 💡 Features

- **Idea → Email**: Type a short idea (e.g., "leave request") and get a full professional email.  
- **Tone Selection**: Choose between **Formal** or **Friendly** styles.  
- **Real-time Generation**: Get instant results while typing.  
- **Copy Button**: Copy generated emails with a single click.  
- **Sinhala → English Email Conversion**: Type in Sinhala, and get a professional English email.  

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, JavaScript (or React/Vue)  
- **Backend**: Node.js / Express (if required)  
- **APIs**: OpenAI API for email generation  
- **Optional**: Translation API for Sinhala → English  

---

## ⚡ How It Works

1. User types a short idea or summary of the email.  
2. AI generates a full professional email based on the selected tone.  
3. User reviews the email and copies it to clipboard.  
4. If input is in Sinhala, AI converts it into English and generates the email.
